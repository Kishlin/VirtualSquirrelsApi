Installation & Usage
--------------------

* Clone this repo.

````bash
git clone https://github.com/Kishlin/VirtualSquirrelsApi.git
cd VirtualSquirrelsApi
````

* Set up dependencies.

````bash
composer install
````

* Start development server.

````bash
php bin/console server:run
````
